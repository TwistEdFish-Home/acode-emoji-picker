# Changelogs

## v1.0.0 — 2026-03-11

- Initial release
- Side button trigger (🤓) with fallback to command palette
- 10 emoji categories: Smileys, People, Animals, Nature, Food, Activities, Travel, Objects, Symbols, Flags
- Search across all categories
- Recently used strip (last 20 emojis, persists per session)
- Inserts at cursor via `editor.session.insert(editor.getCursorPosition(), emoji)`
- Tap-outside-to-close overlay
- Keyboard shortcut: Ctrl+Alt+E
- CSS uses Acode theme variables for native dark/light theme compatibility
